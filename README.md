# Northwind Analytics

Projeto de analytics engineering para o desafio prático da Northwind Traders.

## Objetivo
Construir uma camada analítica para gerar indicadores de performance e apoiar decisões para aumento de ticket médio e redução de churn.

## Stack
- PostgreSQL
- dbt
- Python
- Power BI

## Como rodar
1. Subir postgres com docker compose
2. Carregar csvs
3. Executar dbt seed / run / test
4. Atualizar dashboard / exportar PDF

## Principais entregáveis
- camada analítica em dbt
- documentação de métricas
- dashboard executivo
- relatório final em PDF
